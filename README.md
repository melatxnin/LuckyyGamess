NOM DU JEU
Lucky Games


DÉVELOPPEUR 
Nicolas Gossard


OBJECTIF
Projet réalisé sur le moteur de jeu Unity dans le cadre d'un rendu scolaire ayant pour
perspective principale la structure de données. 

La consigne princiale est d'utiliser dans ce jeu au moins une fois chacun de ces éléments :
- List
- Dictionary
- Queue
- Array
- Stack


PLATEFORME
IOS


VERSION D'UNITY
6000.3.3f1


DESCRIPTION
Ce jeu est un ensemble de mini-jeux, dans un style pixel-art simpliste aux couleurs vives et
contrastées, dont le thème central est le casino.

Le premier mini-jeu apellé "BLACK JACK", est comme son nom l'indique un simulateur de Black Jack
où le joueur joue seul contre le croupier. Les mécaniques sont plus ou moins les mêmes que dans
le vrai jeu, aussi bien dans le comportement du croupier que dans les contraintes du joueur.

Le deuxième mini-jeu appelé "ARKANOID ROYALE", est un Casse-Brique revisité façon machine à sous,
non pas car le joueur doit mettre quelques euros pour pouvoir y jouer, mais car l'objectif principal
est de casser trois briques d'un même symbole afin de profiter de différents bonus.

Le troisième mini-jeu appelé "BACCARAT", est un jeu de Baccarat avec les règles de bases du jeu,
donc également similaire au BLACK JACK en termes de fidélité au jeu réel. C'est un jeu moins
connu que le Black Jack, mais qui sait plaire à beaucoup de personnes pour son côté simple mais intense.


VOIX
Toutes les voix du jeu ont été générées par intelligence artificielle via l'application mobile Inshot.


SPRITES
Tous les sprites du jeu ont été entièrement réalisés par moi et mes quelques compétences sur le logiciel
Aseprite, ce pourquoi ils ne sont objectivement pas très complexes, mais simplement uniques.
